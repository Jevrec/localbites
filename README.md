# Restavracije po mestu — Next.js

Enostavna spletna aplikacija, zgrajena v **Next.js**, ki uporabniku omogoča vnos imena mesta ter pridobitev seznama restavracij v tem mestu.  
Aplikacija uporablja **Google Maps API** in **Google Places API** za iskanje lokacij in restavracij, podatki pa se shranjujejo in berejo iz **Sanity CMS (database)**.

---

## Funkcionalnosti

- Iskanje restavracij po imenu mesta (npr. Ljubljana, Maribor, Paris)
- Pridobivanje podatkov prek **Google Maps & Places API**
- Shranjevanje in branje restavracij iz **Sanity CMS**
- Moderna arhitektura (Next.js 14, React, TypeScript)

---

## Tehnologije

- **Next.js 14** — React ogrodje  
- **React** — uporabniški vmesnik  
- **TypeScript** — statično tipiziran JavaScript  
- **Google Maps API** — geokodiranje mest  
- **Google Places API** — iskanje restavracij  
- **Sanity CMS** — headless CMS / database
